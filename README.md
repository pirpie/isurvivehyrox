# isurvivehyrox 🏃
> A free HYROX training tracker. Built for athletes, by someone cheering from the sideline.

## 🔗 Live App
https://isurvivehyrox.com

## Features
- Track progress for all 8 HYROX stations
- Race day countdown
- Benchmark against 700,000+ real race results
- Progress charts per station
- Dark mode, works on mobile, installable as PWA

## Built With
- React + Vite
- Recharts
- Tailwind CSS
- GitHub Pages

## The Story
My boyfriend is training for HYROX DC. Existing apps were too complex. So I designed and built this in a weekend.
